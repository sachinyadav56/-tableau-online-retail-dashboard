# Online Retail Sales Tableau Dashboard

## Project Overview

This project analyzes an online retail dataset using Tableau to identify sales trends, top products, customer behavior, and geographic sales distribution.

## Dashboards Included

* KPI Summary (Total Sales, Customers, Invoices, Avg Order Value)
* Monthly Sales Trend
* Sales by Country Map
* Top N Products Analysis
* Customer Leaderboard
* Interactive Parameter Controls

## Features

* Dynamic Top N product selection
* Metric toggle (Sales, Quantity, Invoices)
* Interactive filters for Country
* Executive-level dashboard layout

## Tools Used

* Tableau Public
* Excel Dataset
* GitHub
